为什么这版更松

它不再把全部信息挤进一条流程上。
卡片先讲结果 先看“现在走到哪”，再决定要不要下钻看节点细节。
角色和任务拆开 谁负责、下一步是什么、当前卡点是什么，不堆在同一块里打架。
适合正式接入 后面不止爱慕一个 Case 时，这种版式最容易扩成产品首页。
